# ⚡ n8n Automation Workflows

A collection of real-world automation workflows built with n8n
that solve everyday problems for college students and developers.

## 🤖 What is n8n?
n8n is a workflow automation tool that connects apps and services
together — no heavy coding required. Think of it as building
with Lego blocks where each block is an app or action.

## 📁 Workflows

| Workflow | Description | Status |
|---|---|---|
| Internship Scanner | Scans Internshala every 2hrs, sends Telegram alerts | ✅ Live |
| Coming soon... | More workflows in progress | 🔄 |

## 🛠️ Tech Stack
- n8n (workflow engine)
- Telegram Bot API
- JavaScript
- HTML scraping
- REST APIs

## 🚀 How To Use
1. Install n8n on your machine
2. Import any `.json` workflow file
3. Add your own credentials
4. Activate and run!
